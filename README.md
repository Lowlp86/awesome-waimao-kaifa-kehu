## 写在前面：我是谁，为什么搞这份清单

我是 Leo，做 B2B 外贸 + 工厂采购十来年，自己也有个小团队接单。机械配件、家居、小电子都碰过，平台站、独立站、展会、地推都干过。

为什么写这个？两件事戳到我了：

1. **GitHub 上中文几乎是空白。** 英文圈有几份还不错的 awesome 清单（搜 `b2b leads` / `ai lead generation` 那几个 repo），但没人汉化、也不贴合中国外贸人的实际场景（我们要管的是阿里国际站、海关数据、WhatsApp、Google 这一整套，不是纯北美 SaaS 打法）。所以我照着抄了一遍、删掉用不上的、补上我们真正在用的。
2. **行情逼着你换打法。** 这两年明显感觉到：B2B 平台的询盘转化掉得厉害（行业报告口径下大致到 **3.7% 左右**），而获客成本反而**同比涨了快三成（+28%上下）**。⚠️ 这些数字各家统计口径不一样，**看趋势别抠绝对值**——但趋势很清楚：**「在平台等询盘」越来越不划算，得自己主动出去找客户、找对的人、用工具放大效率。**

这份清单就是干这个的。我尽量做到**有褒有贬**——好用的说好用，坑的明说坑，贵的标贵。工具天天在变价、被收购、改套餐，**发现链接挂了或信息过期，欢迎提 PR 或开 Issue。**

> 一句免责：我不收任何工具的推广费，下面没有一条是返佣链接。纯个人踩坑记录。

## 0. 先想清楚：主动获客的漏斗长什么样

工具买之前先有个地图，不然钱花一堆、流程乱成一锅。我自己用的就是这条线：

```
找名单           找联系方式        触达 & 跟进         接得住
─────────       ─────────         ─────────          ─────────
海关数据   ─┐                    开发信(冷邮件)        官网/落地页
LinkedIn    ├─►  邮箱/WhatsApp ─►  LinkedIn 私信   ─►   询盘表单
企业数据库 ─┘    (验证!)           WhatsApp 触达        CRM 接住跟进
                                  展会/红人引流
```

每一段都有专门的工具，下面按这个顺序排。**重点：最右边那一格「接得住」最容易被忽略，但它决定前面所有钱的转化率**，单开一节讲（见下文落地页那段）。

## 1. LinkedIn 开发

欧美、印度、中东很多决策人都泡在 LinkedIn。这是**最适合做「精准找人 + 软触达」**的渠道，但平台反爬严，自动化用猛了容易封号，悠着点。

- **[LinkedIn](https://www.linkedin.com)** — 大本营本身。免费版能搜、能加好友、能发私信，起步阶段把个人主页和公司页做扎实，比买任何工具都重要。
- **[Sales Navigator](https://business.linkedin.com/sales-solutions/sales-navigator)** — LinkedIn 官方的高级搜索，按行业/职位/公司规模/地区精筛决策人，还能存线索、看动态找切入点。贵（个人版月费几十刀），但做 B2B 高客单的，这是地基，绕不开。
- **[Evaboot](https://evaboot.com)** — 把 Sales Navigator 的搜索结果一键导出成干净 Excel，还顺手清洗、找并验证邮箱。省下大把手动复制的时间；按导出量计费，重度用会累加。
- **[Phantombuster](https://phantombuster.com)** — LinkedIn（及多平台）自动化抓取/加好友/发消息的老牌工具，能力强、模块多。**双刃剑**：配置不当 + 频率太猛 = 账号被限，新号别一上来就猛冲。
- **[Dux-Soup](https://www.dux-soup.com)** — 浏览器插件式的 LinkedIn 自动化（自动访问主页、加好友、跟进），上手简单、便宜，适合个人和小团队。重度自动化同样有封号风险。
- **[Waalaxy](https://www.waalaxy.com)** — LinkedIn + 邮件的多渠道触达序列，界面友好、模板多，欧洲团队常用；自由度不如 Phantombuster，但更省心、更安全一点。
- **[SalesQL](https://salesql.com)** — LinkedIn 资料页一键挖个人/企业邮箱的轻插件，免费额度适合零散查人。

> 🔑 LinkedIn 心法：**先把内容和人设做出来再自动化**。一个空白头像、零动态的号去批量加好友，通过率极低还容易被封。我自己是先发 2~3 周行业干货，再开始主动加人。

## 2. 海关数据 / 企业数据

这块是外贸人的「金矿」：**直接看你的目标客户在跟谁买、买什么、买多少、多久买一次**，顺藤摸瓜挖采购联系人，也能扒竞争对手的客户。

- **[ImportYeti](https://www.importyeti.com)** — ⭐ 入门首选，**免费**查美国海关提单，看进口商在跟哪些供应商采购、频次和量级。缺点很明确：**只覆盖美国海运进口**，查不到空运、也查不到其他国家。但作为「先免费摸一遍水」的工具，无敌。
- **[ImportGenius](https://www.importgenius.com)** — 老牌付费海关数据，覆盖国家比免费工具多、数据更细，支持按 HS 编码、公司名深挖。适合已经验证渠道有效、愿意付费上量的团队。
- **[Panjiva](https://panjiva.com)** — S&P Global 旗下，全球贸易数据 + 公司画像，数据权威、分析维度全。偏企业级，价格也偏企业级，小团队慎入。
- **[Volza](https://www.volza.com)** — 主打覆盖国家广（号称几十个国家进出口数据）、性价比相对友好，常被拿来和 ImportGenius 比。数据完整度因国家而异，下单前务必用试用核对你目标市场的数据质量。
- **[腾道 Tendata](https://www.tendata.com)** — 国内厂商，中文界面 + 中文客服，海关数据 + 全球企业库 + 营销触达打包，对不想啃英文工具的团队友好。打包销售、销售跟得紧，自己想清楚需求再谈，别被套餐带跑。
- **[Trade Map (ITC)](https://www.trademap.org)** — ⭐ **免费**的官方贸易统计（国际贸易中心做的），看某个 HS 编码在哪些国家进出口量大、增长快——**选市场、定方向**时极好用。它给的是宏观国别数据，不给具体公司名，和上面的提单数据是互补关系。

> 🔑 用法：先用 **Trade Map** 锁「去哪个国家」，再用 **ImportYeti / 海关数据** 锁「这个国家里有哪些活跃买家」，最后用第 1、第 4 节的工具找到具体的人和邮箱。

## 3. 获客自动化（数据编排 + 工作流）

把「找名单 → 补全信息 → 验证 → 推进 CRM/发信」串成自动流水线。**小团队的杠杆全在这一节**——一个人能干过去一个组的活。

- **[Apollo.io](https://www.apollo.io)** — ⭐ 一体机：**2 亿+ 企业联系人数据库 + 开发信序列 + 拨号**，免费额度大方，性价比之王，预算紧的小团队我一般首推它先用起来。短板要知道：**欧洲/中国本土数据偏弱、邮箱准确率参差、自带发信的送达率不如专业冷邮工具**——重度发信建议数据用它、发信换专门工具。
- **[Clay](https://www.clay.com)** — 当下最强的数据编排/enrichment 平台，能把几十个数据源 + AI 串成一条流水线（自动找人、找信息、写个性化开头）。能力天花板很高。代价是**贵 + 学习曲线陡**，小团队没量、没人专门搭，容易交了学费用不回票价。先想清楚再上。
- **[n8n](https://n8n.io)** — ⭐ 开源、可自托管的自动化工作流（Zapier/Make 的开源平替），数据放自己服务器、能接 API 和 AI 模型，**重度自动化 + 在意数据归属和成本的，强烈推荐**。要点技术，纯小白上手有门槛。
- **[Make](https://www.make.com)** — 可视化自动化，比 Zapier 灵活、按操作量计费往往更便宜，复杂多步流程好搭。
- **[Zapier](https://zapier.com)** — 连接 App 最多、最傻瓜，几乎你用的 SaaS 它都能连。胜在生态和省心，量一大单价就不友好。
- **[ZoomInfo](https://www.zoominfo.com)** — 北美 B2B 数据的「天花板」，数据深、合规做得足，但**很贵、合同年付起跳**，是大公司的玩具，列在这儿是让你知道有这么个标杆。
- **[Lusha](https://www.lusha.com)** / **[Cognism](https://www.cognism.com)** — ZoomInfo 的两个常见平替：Lusha 轻量好上手、插件方便；Cognism 在欧洲数据和合规（GDPR）上口碑好，做欧洲市场可重点看。

## 4. AI 开发信 / 冷邮件工具

⚠️ **冷邮件能不能成，七分在「能不能进收件箱」，三分在「写得好不好」。** 别只盯着 AI 写文案，**送达率（deliverability）才是命门**——这也是 Apollo 自带发信常被吐槽的地方。

- **[Instantly](https://instantly.ai)** — ⭐ 冷邮件圈现在很火的一体方案：**多邮箱轮发 + 自动预热 + 送达优化**，专为「冷启动大量发信」设计，送达率口碑好，价格对小团队友好。配合 Apollo 出数据是经典组合。
- **[Smartlead](https://www.smartlead.ai)** — 和 Instantly 直接对打，主打无限邮箱、自动预热、统一收件箱（unibox）管理回复，重度发信、做代运营的喜欢它。两家挑一个上手即可，别都买。
- **[Lemlist](https://www.lemlist.com)** — 主打**个性化**（自动插入定制图片/视频、多渠道序列），适合走「少而精、高度定制」路线而不是海发的人。
- **[Reply.io](https://reply.io)** — 多渠道销售触达（邮件 + LinkedIn + 电话）老牌选手，AI 功能加得早，序列管理成熟。
- **[Hunter.io](https://hunter.io)** — ⭐ 找企业邮箱 + 验证，免费额度够小团队，浏览器插件好使。深度找人不如 Apollo/Snov，但「补一个邮箱、验一下真假」它顺手又靠谱。
- **[Snov.io](https://snov.io)** — 找邮箱 + 验证 + 冷邮件序列一体，性价比高，常被当 Hunter + 轻量发信工具的合体平替。
- **[NeverBounce](https://neverbounce.com)** — ⭐ 专门的邮箱验证服务，**发信前先洗一遍名单**，把无效/陷阱邮箱剔掉，能实打实救你的发件域名声誉。冷发名单务必先过一遍验证，这步省不得。
- **[Mailmeteor](https://mailmeteor.com)** — 极简的 Gmail 邮件合并群发插件，量小、刚起步、不想买重型工具时够用了，便宜。

> 🔑 三条保命法则：①新域名/新邮箱**先预热 2~4 周**再发；②每个邮箱每天别贪量；③**发前必验证名单**（NeverBounce）。这三条不做，工具再贵也白搭，分分钟域名进黑名单。

## 5. 询盘识别 / 访客追踪 / CRM

流量进来了，得**知道是谁来了、把人接住、跟进别掉链子**。再好的线索，跟丢了等于没有。

- **[HubSpot CRM](https://www.hubspot.com)** — ⭐ **免费版就很能打**的 CRM，管联系人、记跟进、看管道，营销/销售/客服打通。小团队从免费版起步完全够，付费功能按需再加。
- **[Pipedrive](https://www.pipedrive.com)** — 销售管道管理做得最直观的 CRM 之一，便宜、专注「推单子」，外贸 SOHO 和小团队很顺手。
- **[Zoho CRM](https://www.zoho.com/crm)** — 功能全、价格友好，套件生态大（邮件、表单、财务都有），适合想用一家把全套办公搞定的团队。
- **[Leadfeeder（现属 Dealfront）](https://www.leadfeeder.com)** — ⭐ **识别访问过你官网但没留资的公司**（基于企业 IP 反查），把「来看了没说话」的潜在客户挖出来主动跟进。对官网有一定流量的外贸站特别有用；它识别的是「公司」不是「具体某个人」，预期别错位。
- **[Clearbit](https://clearbit.com)** — 企业数据补全 + 访客识别（现属 HubSpot 生态），把你 CRM 里干瘪的联系人补成完整画像。

## 6. WhatsApp / 社媒触达

中东、东南亚、拉美、印度——**很多市场邮件没人看，WhatsApp 才是主战场**。社媒同理，决策人活跃在哪你就得去哪。

- **[WhatsApp Business](https://business.whatsapp.com)** — 官方免费 App，小团队手动维护客户、发目录、设快捷回复，起步够用。规模化要上 API（下面几家）。
- **[WATI](https://www.wati.io)** — 基于 WhatsApp Business API 的客户沟通平台，团队共享收件箱 + 自动化 + 群发模板，中小外贸团队用得多，上手相对快。
- **[360dialog](https://www.360dialog.com)** — 官方认证的 WhatsApp Business API 服务商，按官方价走、不层层加价，适合自己有开发能力、想直接对接 API 的团队。
- **[Twilio](https://www.twilio.com)** — 老牌通讯 API 巨头（短信/WhatsApp/语音全有），稳定、文档全、全球覆盖，适合要把消息能力嵌进自己系统的团队。偏开发向。
- ⚠️ **红线**：WhatsApp 对**未经同意的批量营销打击很严**，乱群发轻则限流重则封号。先拿到对方同意（opt-in）再上自动化，别拿主力号去赌。

> Facebook、Instagram、X 等社媒触达，更适合「内容 + 软触达」长期养，工具因平台政策变动频繁，这里先不堆链接，欢迎 PR 补你验证过好用的。

## 7. 展会 / 红人与 KOL

线下展会和海外红人，是**信任度最高的获客方式**，适合做品牌和高客单。

- **[10times](https://10times.com)** — ⭐ 全球展会数据库，按行业/国家/时间查展会，看参展商、订日程、提前约客户。备展找展会先上这儿。
- **[Trade Fair Dates](https://www.tradefairdates.com)** — 另一个全球展会日历，覆盖欧洲展会尤其全，和 10times 互为补充、交叉核对日期。
- **[Modash](https://www.modash.io)** — 海外网红/KOL 搜索与分析（覆盖 Instagram/YouTube/TikTok），按受众地区、品类筛红人，做品牌出海找带货/测评合作用得上。
- **[Upfluence](https://www.upfluence.com)** — 红人营销全流程平台（找人 + 沟通 + 管理 + 效果追踪），偏成熟品牌、有红人预算的团队。

## ⚠️ 工具买齐了还是没单？先看你的落地页

这节我必须单拎出来，因为它是**我自己交过最贵学费的地方**。

上面的工具，本质都在干一件事——**把对的访客弄到你的网站/落地页上**。可访客点进来那一下，如果：

- 落地页 3 秒还打不开（海外服务器 + 没优化的模板站，太常见了）；
- 手机端排版崩、按钮点不准、图糊；
- 整页找不到一个像样的询盘表单 / WhatsApp 按钮，或者表单一堆必填项把人吓跑；

——**那前面 LinkedIn、海关数据、开发信花的每一分钱，都喂了水漂。** 获客是个漏斗，你拼命往上面灌水，底下漏一个大洞，灌得越多漏得越多。

我早年贪便宜，用某模板平台搭的外贸站，Google 一测 [PageSpeed](https://pagespeed.web.dev/) 移动端常年三四十分，广告点进来跳出率高得吓人，询盘少得可怜。后来我把站彻底重做了一遍（找的 [模酷 ModalCube](https://modalcube.com) 做的 Next.js 站，手机端 PageSpeed 跑到 90+，代码也是直接交到我自己的 Git 仓库），**同样的流量来源、同样的开发信，留资量明显上了一截**。那一刻我才真信「底盘」比「灌水」重要。

所以这份清单专门留这一节提醒你：**别只盯前端进水量，盯漏的那个底。** 上自动化之前，先拿 PageSpeed 测一遍你的落地页，移动端跑分、3 秒首屏、清晰的询盘入口——这三样不达标，先把站修好再谈获客。

> 自查三件套：① [PageSpeed Insights](https://pagespeed.web.dev/) 移动端跑分（争取 80+）；② 手机上自己点一遍询盘流程顺不顺；③ 表单必填项能砍则砍（理想就要个名字 + 邮箱/WhatsApp）。

## 🧭 怎么搭：三种预算下的获客栈

别一上来全买。按阶段来，跑通一段再加一段。

| 阶段 | 一句话目标 | 我会怎么搭（举例，非唯一解） | 月成本量级 |
|---|---|---|---|
| **0 → 1 冷启动** | 先验证「这套打法对我品类有没有戏」 | ImportYeti(免费) + Trade Map(免费) + Hunter(免费额度) + HubSpot 免费 CRM + 手动 WhatsApp Business | 几乎 0 |
| **1 → 10 跑量** | 把跑通的流程上量、自动化 | Apollo（数据+找人）+ Instantly/Smartlead（发信预热）+ NeverBounce（验证）+ Sales Navigator（LinkedIn 精筛）+ Pipedrive/HubSpot 跟进 | 中等 |
| **10 → N 规模化** | 多渠道编排、把人效拉满 | Clay / n8n（编排）+ 付费海关数据（ImportGenius/Volza）+ WATI（WhatsApp 规模化）+ Leadfeeder（访客识别）+ 专人维护 | 较高 |

> 🔑 三条通用建议：①**先免费跑通再花钱**，别用买工具代替想清楚打法；②**数据工具和发信工具分开**，各用各的强项（数据找 Apollo、发信找 Instantly/Smartlead）；③**先修落地页再灌流量**（见上一节）。

## 🕳️ 避坑清单（血泪版）

- **❌ 平台询盘当唯一来源** → 平台转化在跌、成本在涨，把自己活成「等饭吃」，旺季还好淡季要命。主动获客的渠道至少铺 2~3 个。
- **❌ 新域名/新邮箱直接猛发冷邮件** → 没预热就海发 = 域名进黑名单，老域名都可能被连累。**先预热 2~4 周，发前用 NeverBounce 验证名单。**
- **❌ 海关数据当万能** → 提单数据告诉你「谁在买」，但**给的常是公司/进口商，不直接给采购决策人的邮箱**，还得配 LinkedIn / 邮箱工具去找到具体的人。免费工具（如 ImportYeti）只覆盖美国，别拿它判断全球。
- **❌ LinkedIn 新号一上来就批量加人发消息** → 大概率限制/封号。先养内容、养人设，自动化频率压低，慢就是快。
- **❌ WhatsApp 拿主力号批量群发陌生人** → 违反平台政策，封号没商量。务必先拿 opt-in。
- **❌ 买了一堆工具，没人把流程串起来** → 工具不等于结果。先有「找名单→验证→触达→跟进」的 SOP，再按 SOP 配工具，别反过来。
- **❌ 只优化获客前端，落地页烂着不管** → 漏斗底下漏水，灌越多漏越多（详见落地页那节）。
- **❌ 名单不验证就发 / 不清洗就拉进 CRM** → 脏数据进系统，越用越乱，送达率和跟进效率全拖垮。
